Task 2: Temperature Converter

Name: Pooja Bhosale

Description:
A Java-based Temperature Converter that converts temperatures between Celsius and Fahrenheit.

Features:
1. Celsius to Fahrenheit Conversion
2. Fahrenheit to Celsius Conversion
3. User Input using Scanner
4. Mathematical Conversion Formulas
5. Clear and Accurate Results

Conversion Formulas:
Fahrenheit = (Celsius × 9/5) + 32
Celsius = (Fahrenheit - 32) × 5/9

Language Used:
Java

Tools Used:
- Java
- Scanner Class
- Methods
- Conditional Statements

Outcome:
Successfully converts temperatures between Celsius and Fahrenheit based on user choice.
